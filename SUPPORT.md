# Support

Kizomi is a community-maintained, non-commercial hobby project with no guaranteed response time.

Use the repository's GitHub Issues page for reproducible bugs, accessibility problems, provider-integration regressions, and documentation corrections. Include the app version, Android version, selected provider, expected result, actual result, and safe reproduction steps.

Do not publish access tokens, refresh tokens, authorization codes, PKCE values, callback URLs, client secrets, account identifiers, private notes, full provider responses, or screenshots containing private information.

For suspected security vulnerabilities, follow [SECURITY.md](SECURITY.md) rather than opening a public issue with exploit details.
